# 💰 SpendWise – Expense Tracker System

A simple and modern expense tracker web application built using **PHP, MySQL, JavaScript, and Chart.js**.
This project helps users manage their income and expenses while providing visual summaries through interactive charts.

---

## ✨ Features

* User Authentication (Login & Register)
* Add Income and Expenses
* Transaction Categories
* Edit & Delete Transactions
* Monthly Financial Summary
* Interactive Charts using Chart.js
* Responsive Dashboard UI
* MySQL Database Integration

---

## 🛠 Technologies Used

* PHP
* MySQL
* JavaScript
* AJAX
* Chart.js
* HTML/CSS

---

## 📊 Dashboard Overview

Users can:

* Track total income and expenses
* Monitor remaining balance
* View categorized spending
* Analyze monthly financial activity through charts
